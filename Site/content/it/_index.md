## Cos'è DutyPusher?

Questo è un plugin **Final Fantasy XIV** per Dalamud.

Quando attivi questo plugin e configuri un canale di notifica, invierà una notifica al tuo dispositivo quando una coda per un dungeon è pronta nel gioco.
